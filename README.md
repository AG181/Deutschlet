# Deutschlet
Deutschlet - это простое и эффективное приложение для изучения немецкого языка с помощью флеш-карточек и встроенных тестов. Программа написана на Python с использованием библиотеки PyQt6.

Функции
Флеш-карточки: Удобный интерфейс для запоминания слов (немецкий — русский).

Тестирование: Автоматическая проверка знаний по категориям (глаголы, существительные, прилагательные).

Гибкое управление: Возможность перемешивания слов, навигация между карточками и сброс прогресса.

Современный дизайн: Интерфейс выполнен в темной теме с использованием шрифта Roboto.

Структура данных
В программе реализованы три основные категории:

Глаголы: Более 100 базовых глаголов для ежедневного общения.

Существительные: Более 100 популярных существительных с артиклями.

Прилагательные: Набор базовых описательных слов.

Технологический стек
Язык: Python

GUI: PyQt6

Сборка: PyInstaller
